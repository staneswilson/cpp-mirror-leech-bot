# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. The full text is published at the link above and applies in full.

## Scope

The Code of Conduct applies in all project spaces — issues, pull requests, discussions, code review comments, the project chat, and any other forum where contributors engage on behalf of the project. It also applies when an individual is officially representing the project in public spaces (e.g., using the project's official email or social media accounts).

## Enforcement contact

Reports of behaviour that may violate the Code of Conduct should be sent privately to the project maintainers via a [GitHub Security Advisory](https://github.com/staneswilson/cpp-mirror-leech-bot/security/advisories/new) (the same private channel used for security disclosures), or by DM to the GitHub user [@staneswilson](https://github.com/staneswilson). Reports are handled confidentially and reviewed within five business days.

When reporting, please include:

- A description of the incident and where it occurred
- The accounts involved (if applicable)
- Any supporting context (screenshots, logs, links)
- Whether you have concerns about retaliation

## Enforcement guidelines

Project maintainers follow the four-stage enforcement ladder published in Contributor Covenant 2.1 (Correction → Warning → Temporary Ban → Permanent Ban). The action taken in response to a confirmed violation depends on severity, prior history, and the maintainers' assessment of risk to the community.
